#Steps to obtain tidy dataSet
1. Clone the repo `git clone https://github.com/Sephiroth-XIII/GettingAndCleaningData.git`
2. Open the repo `cd GettingAndCleaningData`
3. Run the rscript `Rscript run_analysis1.R`
4. Tidy Data Set will be inside `UCI HAR Dataset` directory in a file named `averaged_tidy_data_set.txt`
